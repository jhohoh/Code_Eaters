## 변수명 규칙
1. 변수명이 길어지는 것에 두려워 말고 의미 있는 변수명으로 선언
2. 변수명은 ‘카멜케이스’로 선언
3. 함수 (메서드) 의 로직은 되도록이면 5~6줄이 넘어가면 안된다
4. 주석은 무조건 주석을 달고 싶은 해당 본문의 위에 적자
5. 파일마다 첫째줄에는 해당 파일에 무슨 내용이 있는지 간략히 요약하기
